# Particle Transport and Collision

Use for event generation, detector geometry transport, collision
studies, cross sections, histograms, ROOT-facing analyses, and
high-energy or particle-physics workflows.

## Open-Source Defaults

- PYTHIA: event generation and generator-level studies.
- MadGraph5_aMC@NLO: matrix elements and event generation handoff.
- Geant4: detector/geometry/material transport.
- ROOT: analysis, histograms, and artifact inspection.

## Validation

Track generated events/histories, rejection or error rate, random seed,
cross-section or tally progress, conservation checks, geometry/material
warnings, and output file growth. Preserve seeds and cards.

# Athena++ Handoff Checklist

- [ ] Record authorized execution host, compiler flags, and Athena++ version.
- [ ] Confirm problem generator, units, mesh, boundaries, and time limit.
- [ ] Run low-resolution smoke before any production mesh.
- [ ] Capture stdout/stderr, history output, checkpoints, and plot paths.
- [ ] Parse simulation time, redshift if applicable, conserved drift, checkpoint freshness, and load-balance warnings.
- [ ] Attach dashboard/result/failure Note only after execution evidence exists.

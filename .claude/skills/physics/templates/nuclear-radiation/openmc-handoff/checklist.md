# OpenMC Handoff Checklist

- [ ] Confirm authorized cross-section library path without exposing secrets.
- [ ] Fill geometry, materials, source, and tallies.
- [ ] Run low-history smoke before production histories.
- [ ] Capture stdout/stderr, statepoint metadata, tally table, and entropy output.
- [ ] Parse histories, relative error, Shannon entropy, and missing data warnings.
- [ ] Attach dashboard/result/failure Note only after execution evidence exists.
